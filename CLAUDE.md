# CLAUDE.md

Este arquivo será gerado pela skill `/comece-por-aqui` com a identidade da sua IA personalizada.

Para documentação do framework, veja [CLAUDE-IC.md](CLAUDE-IC.md).

## Estrutura do projeto

```
memoria/          ← Memórias da IA (no .gitignore)
estudos/          ← Resumos pessoais de estudo (no .gitignore)
troca/            ← Troca de arquivos pessoais (no .gitignore)
guias/            ← Guias públicos do framework
modelos/          ← Templates
exemplos/         ← Exemplos sanitizados
.claude/skills/   ← Skills do projeto
```

## Propósito

Este repositório é o **framework de Interface de Colaboração com o Claude Code** — identidade, memória, skills e troca de arquivos. Não contém código de produto; é a infraestrutura que torna a colaboração persistente entre humano e IA.
